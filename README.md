# ainfo.dev - Top 10 Open Source LLMs

A premium static site showcasing the top open-source Large Language Models of late 2024 and 2025.

## Features

- **Premium Design**: Dark mode, glassmorphism, and smooth animations.
- **Detail Pages**: In-depth specs, "Hello World" code examples, and industry usage for 10 top models.
- **SEO Optimized**: Fully optimized with Open Graph tags, Sitemap, and Robots.txt.
- **Responsive**: Looks great on all devices.

## Models Covered

1. Llama 3.1
2. DeepSeek V3 & R1
3. Mistral Large 2
4. Qwen 2.5
5. GPT-OSS
6. Gemma 2
7. Phi-4
8. Grok-1
9. Yi-1.5
10. Command R+

## Deployment

This site is ready for deployment on GitHub Pages, Vercel, or Netlify.
